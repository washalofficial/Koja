This repo contains an Expo React Native frontend and a Node/Express backend using MongoDB.

## Quick start

### Frontend
1. cd frontend
2. npm install
3. Add Firebase config in src/services/firebase.js
4. npm run start

### Backend
1. cd backend
2. npm install
3. Create a .env with MONGO_URI and JWT_SECRET
4. npm run dev (or node server.js)
